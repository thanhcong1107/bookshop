# Bookshop
## 1.Description function
This website has two users: `customers` and `admin`
### 1.1 Customers
- Login/Register
- Edit profile
- Fine a book, view detail
- Add a book to cart/wishlist
- View cart, order
- Payment: internet banking or ship COD
- Write review, rate a book
### 1.2 Admin
- Manage product
- Manage category
- Manage author
- Manage publisher
- Manage order
